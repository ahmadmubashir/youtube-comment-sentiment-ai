# youtube-comment-sentiment-ai
YouTube comment sentiment analysis using HuggingFace Transformers and OpenAI GPT with Streamlit dashboard and AI-generated insights.
# 📊 YouTube Comment Sentiment Analyzer + AI Insights

This project analyzes YouTube video comments using NLP and generates AI-powered insights.

---

## 🚀 Features

- Fetch YouTube comments via API
- Sentiment analysis using HuggingFace Transformers
- Visualization (Bar + Pie charts)
- AI summary using OpenAI GPT
- CSV download of comments
- Interactive Streamlit dashboard

---

## 🧠 Tech Stack

- Streamlit
- HuggingFace Transformers
- Google YouTube API
- OpenAI GPT (LangChain)
- Pandas, Matplotlib
